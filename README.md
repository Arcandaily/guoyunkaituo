## 快速开始

### 1. Obsidian 知识库
1. 安装 [Obsidian](https://obsidian.md/)
2. Open folder as vault → 选择本仓库的 `obsidian_vault/`
3. 从 `00-索引/00_MOC.md` 进入

### 2. 数据表（Excel）
| 文件 | 用途 |
|---|---|
| `data/国运开拓_50国_网友代称与关系.xlsx` | 50 国代称、大洲、对华/灯塔/欧盟关系、渊源 |
| `data/开拓者数据面板_追踪.xlsx` | 每人属性/资源/蓝图/流水/国运评分/直播开拓币 |
| `data/舰船数据总表.xlsx` | 200+ 舰船索引与属性（Wiki 整理） |

重新生成：
```bash
pip install openpyxl requests beautifulsoup4
python scripts/build_all.py
python scripts/scrape_ships.py          # 需可访问 biligame Wiki
python scripts/enrich_ships_from_local.py
```

## 小说硬设定（摘要）
- **蓝星** 触达考核临界点 → **人联**（泛星际-人类联盟）拉入考核。
- 每国 **10 名开拓者**；华国因主角 **第 11 席**特例。
- **10 卷考核**：新手 → 考古碎片 → 天灾撤离 → 晶体 → 沦陷模拟 → 星盗 → 智械 → 晶体2.0 → 收复 → 真实前线。
- 目标：华国成领袖阵营，太阳系升维搬迁入 **万族战场** 人联星域。
- 资源：金属 / 晶体 / 重氢 / 电子货币 + 原创 **开拓币**（直播打赏/任务/成就）。
- 主角金手指：修仙炼药炼器思路 × 生物/机械/星舰科技（低灵环境限制）。
