# 开拓者双天赋体系（1+1）

> **硬设定**：人联体系内的开拓者，是能在万族战场与 **邪神、虫族、神族、亚空间恶魔** 等相抗衡的高维文明战斗单元。  
> 本土人联开拓者 **另称「指挥官」**；蓝星考核新血入局后同样纳入开拓者序列，走同一套 **1+1 双天赋**。  
> **天赋三态**：**可升级 · 可交易 · 可产出** → 专页 [[天赋升级交易与产出]]。

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 用户硬设定 | 1+1 天赋；全员【紧急避险】；第二槽按个人特质；**升级/交易/产出** |
| 拉格朗日 | 基地被打爆/迁移、指挥官操作感（机制层） |
| 战锤/SC/群星 | 高维战场、光环、特种能力感（机制层，正文用代称） |
| 本库 | L0 现实身份 → 特质槽概率表；[[执照与编制晋升]] |

## 称谓

| 称呼 | 谁 | 备注 |
|------|-----|------|
| **开拓者** | 人联认证的开拓主体（含蓝星考核生、本土人联） | 考核星系内统一身份 |
| **指挥官** | **本土人联** 开拓者的惯称 | 直播观众、前线编制口语；蓝星新人一般不先叫这个，晋级后可被观众喊「准指挥官」 |
| 见习/执照档 | 见 [[执照与编制晋升]] | 天赋与执照 **正交**：有天赋≠高执照 |

## 为何人联开拓者「扛得住万族」

人联不是普通行星文明：开拓者被纳入 **高维战斗协议**（天赋 + 科技 + 编制）。  
单人仍非无敌——邪神/虫潮/亚空间可碾压 **未编队、未升本、天赋未成长** 的个体；抗衡靠的是：

1. **双天赋**（本页）  
2. **舰船/基地/升本**（[[基地1-10本升级路线]]）  
3. **编制与协同**（国运编队、人联后勤）  
4. **修科等稀缺解**（主角）  

> 写作：人联观众侧可狂傲；前线实体仍在流血（[[机制-人联]]）。

## 1+1 结构

```
槽 0  基础天赋（全员固定）  →  【紧急避险】
槽 1  特质天赋（一人一系）  →  按个人特质/经历/职业倾向觉醒
```

| 槽 | 名称 | 是否人人相同 | 成长 / 经济 |
|----|------|--------------|-------------|
| **0** | 基础天赋 | **是**，模板统一 | **可升级**；微量**产出**（残渣/数据）；**本体禁交易** |
| **1** | 特质天赋 | **否**，按人 | **可升级**；主**产出**源；产物/残片/残响**可交易**（本体禁永久卖） |

**禁止**：第三人开挂槽（除非终局剧情「人联实验体/禁忌觉醒」，需单独机制卡）。  
**主角例外感**：不靠第三槽，靠 **特质天赋品质/修科耦合/产物溢价**（见下）。  
**经济环**：升级 ←素材— 产出 → 交易 → 再升级，详见 [[天赋升级交易与产出]]。

## 槽 0 · 基础天赋【紧急避险】

### 效果（写作版规则）

1. **触发条件**：己方 **主基地**（或考核认定的「家」锚点）即将被摧毁的前一瞬间（结构值归零判定前）。  
2. **锁血**：基地进入极短 **无敌/锁血帧**（叙事：力场冻结、协议接管），避免当帧爆掉。  
3. **瞬移**：基地（及协议绑定的核心设施/部分停泊舰，细则可卷内规定）被挪到 **本星系内随机合法坐标**。  
4. **代价/限制**（必写，防无脑）：
   - **长冷却**（如一赛季有限次数，或 24–72 小时叙事冷却）  
   - **随机落点**可能更差（矿贫、敌多、辐射带）  
   - **前哨/外矿/计划圈** 默认不跟着走（丢外围）  
   - 连续靠避险苟活 → **A/B 评分观感变差**（人联要战友不是永动机）  
   - 亚空间重度污染、协议锁死区可能 **抑制触发**（V10 高压）

### 戏剧功能

- 防止「第一章被偷家直接退赛」的烂尾  
- 制造 **绝境翻盘** 与 **落点惩罚** 双戏  
- 直播名场面：锁血闪现全网弹幕  

### 与游戏体感

对标「基地没了还能再起」的开拓感，但 **有次数与随机性**，不是无限读档。

## 槽 1 · 特质天赋（个性觉醒）

### 觉醒逻辑

```
个人特质（兴趣/职业/执念/创伤/信仰/修炼底色）
        ↓ 人联开拓协议扫描 + 考核高压催化
        ↓
  概率池中抽出 1 个主天赋（可有隐藏保底/国运干预）
```

- **不是** 玩家点技能树随便选；是「像你」的能力。  
- **L0 现实身份** 是最强先验：医生池偏治疗，军人池偏战斗……  
- **高匹配** 提高对应池权重；也可能黑马（教师觉醒战场洞察等）。

### 池与示例（可扩展）

| 特质倾向（觉醒前） | 天赋方向（例名，可改） | 效果语感 | 易适配 L0 |
|--------------------|------------------------|----------|-----------|
| 军事/对抗/射击游戏 | **战意光环 / 火力统合** | 周围友军射速/命中/士气↑ | 军人、特种、警察 |
| 医学/救护/护理 | **救治光环 / 稳态力场** | 伤势恶化减缓、急救成功率↑ | 医生、护士 |
| 工程/修理/动手 | **抢修协议 / 结构再生辅** | 舰船/建筑维修速度↑ | 工程师、矿工、电工 |
| 指挥/组织/带队 | **统御节点 / 指令增幅** | 指挥值效率、集结速度 | 军官、企业家、教师 |
| 侦察/好奇/侦探 | **识破 / 战场透视弱** | 伏兵、隐形、资源异常提示 | 记者、刑警、黑客 |
| 科研/实验 | **解析加速 / 蓝图亲和** | 研究、调校、遗迹解读 | 科学家、程序员 |
| 信仰/静心/戒律 | **镇心 / 抗侵染** | 亚空间幻视抗性、士气崩抑制 | 道士、神父、僧 |
| 物流/生意/算计 | **周转 / 仓储锚定** | 运输损耗↓、交易窗口 | 供应链、贸易、毒枭物流（险） |
| 艺术/表演/误导 | **伪装场 / 注意力诱饵** | 电子/光学欺骗 | 魔术师、演员、网红 |
| 灰黑/生存 | **绝境嗅觉 / 阴影步** | 逃跑线、黑市价、背刺预警 | 帮派、难民（B4 风险） |
| **修仙残留+道士**（主角） | **见下「主角特质」** | 修科耦合 | [[主角-破庙道士]] |

> 表中名为 **写作占位**；定稿可统一「协议代号 + 俗称」。

### 光环类规则感

- 默认 **友军/己方编制** 受益；对他国是否生效看协议与外交  
- 有 **半径 / 人口上限 / 耗能**  
- 可与科技、舰船模组 **叠乘**，但diminishing（防崩）  
- 人联编制夜看：特质是否 **可编入军团模板**（B 轨）

### 成长 · 产出 · 交易（摘要）

| 阶段 | 表现 |
|------|------|
| 初醒 | 弱、不稳定、误触；产出稀少 |
| 考核中期 | 稳定光环；**日产/触发产**进公库；残片开始流通 |
| 高执照/前线 | 进阶名；可萃取**残响**交易；产量够养一支辅队 |

完整规则（Lv/品质、素材、可交易清单、残响衰减）：**[[天赋升级交易与产出]]**。

## 蓝星考核生 vs 本土指挥官

| 项 | 蓝星开拓者（考核） | 本土人联指挥官 |
|----|--------------------|----------------|
| 称呼 | 开拓者（主） | 指挥官 / 开拓者 |
| 双天赋 | **有**，入考核协议后激活/觉醒 | **有**，成熟 |
| 紧急避险 | 有（新手可能不熟冷却） | 用得更刁钻 |
| 特质槽 | 常在 V01 觉醒戏 | 早已定型，可隐藏底牌 |
| 观众观感 | 「原初星球新血」 | 「正规军」 |
| 战力锚 | 科技+天赋+国运组织 | 科技+天赋+编制经验 |

**觉醒时机（建议）**：

1. 签署开拓协议 / 降落考核星系时 **槽 0 写入**  
2. 首次极端压力或「觉醒检定日」 **槽 1 显现**（V01 前 20% 章可写）  
3. 主角可因修仙底蕴 **觉醒偏慢或形态异常** → 直播争议+巨企兴趣

## 主角（破庙道士）特质天赋方向

不设第三槽。建议 **槽 1** 走「道术—工程」耦合，例如：

| 草案名 | 效果 | 与金手指 |
|--------|------|----------|
| **器道共鸣** | 亲手参与改装的模组/药剂 **额外稳定率**；小范围工艺光环 | 直接喂 [[金手指-修科融合]] |
| **清规镇心** | 抗亚空间、稳军心（弱治疗/弱净化） | 道士 L0 + 信仰池 |
| **炉火洞察** | 看穿材料/损伤「火候」 | 工程+炼器 |

定稿时 **三选一为主，其余作成长分支或 equip 式进阶**，避免全能。

## 写作红线

1. 紧急避险 **不能** 每战都放 → 写冷却与代价。  
2. 特质天赋 **不能** 人人满配六维光环。  
3. 本土指挥官强，但 **不是** 无剧情杀全场；蓝星新血用组织与修科翻盘才有爽点。  
4. **不能** 靠交易买来第三槽或永久偷别人本体天赋（常规市场）。  
5. 产出有软顶；升级有稳固期。  
6. 正文少堆系统面板，用 **一次触发 / 一单买卖 / 一次升级** 展示。
