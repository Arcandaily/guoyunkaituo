# 泛星际-人类联盟（人联）

- **定位**：高等人类文明；蓝星考核发起方；守序阵营之一；**万族战场** 人类侧主力编制来源。
- **能力**：定向帮扶、太阳系升维搬迁、考核星系布置、前线直播网、**开拓者双天赋协议**。
- **限制**：万族战场承压，需要新血与「可验证的国运」；人联 **非无敌**，前线在流血。
- **与开拓币**：人联（及人族友军）高级观众的打赏/任务结算为开拓币权威来源之一。

## 开拓者 / 指挥官

| 概念 | 说明 |
|------|------|
| **开拓者** | 人联高维战斗协议下的开拓主体；能与邪神、虫族、神族、亚空间恶魔等 **相抗衡的编制单元**（靠天赋+科技+协同，非个人无敌） |
| **指挥官** | **本土人联** 开拓者的惯称；蓝星考核生一般称开拓者，高表现后观众可能喊「准指挥官」 |
| **双天赋** | **1+1**：全员【紧急避险】+ 特质槽；且天赋 **可升级、可交易（产物/残响）、可产出** → [[开拓者双天赋体系]] · [[天赋升级交易与产出]] |

## 考核哲学（可写进旁白）

> 不是选最可怜的文明，而是选 **在绝境中仍能组织、创造与守序** 的文明。  
> 新血必须证明：接上 **双天赋与舰队** 之后，仍肯为守序而战，而不是只会闪避。
