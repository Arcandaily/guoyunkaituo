# 蓝星冷战格局（平行史）

> **硬设定**：苏联 **并未解体**；冷战 **延续至现代**；华国长期承受 **巨大外交与安全压力**。  
> 目的：给「线乙·蓝星反馈」提供持续张力——开拓考核不是真空争霸，而是 **冷战后延世界的破局钥匙**。

## 华国初始压力阶梯（卷1 前史）

| 层级 | 压力类型 | 例子（代称） | 对应卷次反馈弧 |
|------|----------|--------------|----------------|
| L1 周边小国挑衅 | 岛礁/航道/民粹反华 | 越鸡、菲佣国、泡菜国、长条国 等局部摩擦 | **V01–V02** 正向反馈后降温或反转 |
| L2 区域强国对冲 | 军演/制裁/供应链脱钩 | 霓虹国、袋鼠国、天竺国 等 | **V03–V05** 对抗升级再被开拓成果压回去 |
| L3 超级大国围堵 | 技术封锁、金融、太空轨道话语 | 灯塔国主导；苏系另开条件 | **V06–V08** 话语权实质提升 |
| L4 规则改写 | 蓝星秩序议程设置权 | 联合国式机构话语、人类共同叙事 | **V09–V10** → **人类命运共同体** 落地感 |

## 冷战延续带来的考核特殊规则感

1. **选人更「国运化」**：各国选人掺杂意识形态审查，不纯看能力。  
2. **蓝星支援包带政治标签**：灯塔可给开拓者「盟友援助」但要听话；华国支援更「举国但克制」。  
3. **直播在蓝星是情报战**：同一场矿战，三极媒体三套叙事。  
4. **苏系态度**：战场上可能与华国联手，蓝星谈判桌上要价。

## 线乙爽点总弧（全书）

```
高压窒息（开局）
  → 小胜止谤 / 周边消停（早期卷）
  → 硬刚区域强国（中卷）
  → 国际议程能发声（后中卷）
  → 人类命运共同体从口号变「搬迁后的共同宪章雏形」（终卷）
```

对齐用户目标：**真实地球中国「人类命运共同体」构想**，在小说里用 **考核正向反馈 + 升维搬迁** 实现。
