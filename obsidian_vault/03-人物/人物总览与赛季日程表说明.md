# 人物总览与赛季日程表 · 使用说明

> **文件**：仓库根目录 `data/人物总览与赛季日程_规划表.xlsx`  
> **性质**：只提供结构与空行，**不预填剧情正文**；内容由作者横向/纵向自行维护。

## 和旧表的关系

| 文件 | 侧重 |
|------|------|
| `data/开拓者数据面板_追踪.xlsx` | 数值流水、蓝图、舰队、国运评分（偏「账本」） |
| **`data/人物总览与赛季日程_规划表.xlsx`** | 人物档案 + **每赛季每日写什么**（偏「规划看板」） |

可并存；ID/代号建议两边统一。

---

## 工作表结构

| 表 | 形态 | 你怎么用 |
|----|------|----------|
| **00_使用说明** | 文字 | 总览 |
| **01_人物总览常驻** | 一人一行，列=字段 | 建人设、当前状态；**横向加列**也可（在备注旁插） |
| **02_人物横向赛季列** | 一人一行，向右 V01…V10 块 | **赛季末快照**横向填（战友分/天赋Lv/高光…） |
| **03_赛季目录** | 十季元数据 | 天数、章数上下限、三线焦点 |
| **04_赛季日程总表** | 长表：赛季+日序 | **主推荐**：筛 V01 看 D1–D50；三线分列写提纲 |
| **05_日程_宽表模板** | 字段×日 矩阵 | **一眼扫完整季**；可复制成 V02_宽表… |
| **06_人物日程交叉** | 日×人 细账 | 同一天 10 人各自干嘛（可选） |
| **07_字段字典** | 字段释义 | 防遗忘 |
| **08_复制区_说明** | 操作提示 | 宽表复制法 |

---

## 推荐工作流

```
01 建好人（华国10 + 需要的他国）
    → 03 填本季天数/章数目标
    → 04 或 05 按日填三线提纲（关键词即可）
    → 写章时勾「完成」
    → 季末在 02 横向补快照
```

### 04 长表 vs 05 宽表

| | 04 赛季日程总表 | 05 宽表模板 |
|--|----------------|-------------|
| 形状 | 每行=一天 | 每列=一天 |
| 优势 | 筛选、排序、透视、十季一表 | 横向扫完整季节奏 |
| 适合 | 日常维护 | 排期总览、贴墙感 |

---

## 预置空结构（非剧情）

- `04`：已生成 **V01–V10 × 日序 1–50** 共 500 行，**内容列为空**。  
- 实际一季 40–45 天：多出来的日序空着或删。  
- 超过 50 天：在表尾追加行，日序 51+。

## 列设计对齐本库设定（可选填）

- 三线：线甲开拓 / 线乙蓝星 / 线丙直播  
- 天赋：升级·交易·产出列  
- 双轨：A 国运 / B 战友  
- 国家：用代称（见 [[国家总览]]）  
- 前期高压：冲突类型可写 抢矿/偷家/舆论/内讧/避险 等（见 [[前期高压节奏设计]]）
