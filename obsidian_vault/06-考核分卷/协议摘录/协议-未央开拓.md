# 协议摘录 · 未央开拓（含 3.0 差分）

## 灵感来源（必填）

- Wiki：未央开拓协议、未央开拓 3.0 协议（本库 `wiki_data/protocol_raw/` 有 wikitext）
- 发布方设定：安东尼奥斯开拓者发展部（游戏文案）
- 提取：星门接入、援助舰、评定首要标准、占有率档、未央积分、3.0 计划经营

## 玩法特色（Wiki 提示）

**占有率结算**、**星门建设竞速**。

## 条款要点（改写压缩，非全文复制）

### 开拓事宜

- 签署方经开拓者星门进入目标星系。
- 须遵守《未央公约》中星门归属与使用权相关条款。
- 星门建成并接入拉格朗日系统后，评定结束可通过星门返回枢纽星系。

### 援助（1.0 条款）

- 势力可携带一定数量 **FG300 多功能/多用途护卫舰**。
- 蓝图授权示例：FG300 系、**X8 小型 / X10 中型 / X20 大型工程舰**。

### 评定（核心 → 国运 A1/A2）

Wiki 原文逻辑：

1. **首要标准**：是否拥有对**目标节点星门附属区域**的控制权；控制者获评定积分。  
2. **空间站控制 / 星系占有率**：占有率达档可获对应评定积分。

### 评定称号与门槛（写作可用档位感）

**拉格朗日开创者**（控制星门系）示例：

| 条件感 | 占有率门槛（Wiki） |
|--------|-------------------|
| 高档 | 占领星门空域 + 占有率 ≥40%（3.0 另强调本星系星门不被外星系控制） |
| 中档 | 星门空域 + ≥30% |
| 低档 | 星门空域 + ≥10% |

**区域治理者**（占有率系，可不强调「已占星门」）：

| 档 | 占有率 |
|----|--------|
| 高 | ≥50% |
| 中 | ≥30% |
| 低 | ≥10% |

**开拓先锋**（3.0 强化）：按**经营满级计划数量**分档奖励（计划经营玩法：自动采集、防卫警戒、曲率加速、空域改造等，满级可转建筑计划；单次开拓满级计划相关未央积分有上限描述，公开资料提及至多约 200 点量级——写作用「计划经营贡献」即可）。

### 未央积分

- 按联合体内占有率名次等发放（如开创者官员档可见 1500 / 900 / 450 等量级）。
- 小说映射：**卷末国运积分池**，非开拓币。

### 章节态势（写作节奏素材）

典型推进：清私掠 → 探空域/星门情报 → 贸易与联络站 → 资源与工程扩张 → 星门研究/共振装置 → 格局冲突 → **星门归属终局**。

第一章条件示例：消灭星系内一定数量等级 3 以上私掠舰队。
