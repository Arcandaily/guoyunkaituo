# 协议摘录 · 特洛伊晶体收集

## 灵感来源（必填）

- Wiki：特洛伊晶体收集协议（`wiki_data/protocol_raw/特洛伊晶体收集协议.txt`）
- 玩法特色提示：**挖掘运输**
- 提取：晶体开采、护航防劫、接收站提交、积分=提交量、星门控制+排行

## 背景机制（压缩）

- 特洛伊晶体为合成 **688 号元素** 的原料；688 用于拉格朗日系统运转/高能曲率相关工业（游戏设定）。
- 发包逻辑：经验开拓者签署开采协议，前往富矿星系，保障规模化、持续供给。

## 条款要点

### 开拓事宜

- 开放指定开拓者星门（文案示例 L039）与富矿星系通行。
- 评定后，运输舰队将开采晶体经星门运回枢纽，开拓者再返回。

### 开采与运送约定（→ A1 交付）

1. 尽可能多开采特洛伊晶体。  
2. 保证运输与基地库存**不被抢夺**。  
3. 加入小队后，须运至指定空间站附近的 **特洛伊晶体接收站** 提交。  
4. **提交量 = 所属小队对本次开拓的贡献（评定积分）**。  
5. 未提交部分可自用或交易。

### 空间站开采

- 站管方开采计入**归属势力**提交。

### 评定

1. 接收站提交 → 实时记入小队评定积分。  
2. 空间站采集按速度换算归属势力积分。  
3. 评定时刻按小队总积分排行。

### 称号档（写作）

**特洛伊晶体丰收者**（高阶，常绑星门）：

- 占领星门空域 + 评定积分 ≥1000 且排行第 1/2/3/4–6 等档。

**采集贡献者**（可无星门绑定，偏纯积分榜）：

- 积分 ≥1000 + 排行档。

另有开拓先锋类收尾档（与未央类似）。

## 章节态势素材

- 富矿抵达 → 小行星带找矿 → 竞争与私掠伏击 → 接收站与航线 → 贸易繁荣 → 矿尽争夺激化 → 提炼便于储存（**就地加工**）→ 星门终局与满载返回。

→ 直接支撑小说 **V04 采集** 与 **V08 加工 2.0**。

## 小说用法

| 游戏 | 小说 |
|------|------|
| 特洛伊晶体 | 特种晶体（可保留特洛伊作人联学名） |
| 688 元素 | 曲率/星门维稳原料（勿科普翻车） |
| 接收站提交 | A1 协议交付；被劫 = A3/A4 事件 |
| 积分排行 | 国运榜；护航专家涨 B1/B5 |

## 禁止事项

- 勿把晶体写成万能灵石；保持工业原料定位（与主角修科融合另开特例通道）。
