## 1. 星系地理（可写场景）

> 物理层用真实航天常识；玩法层对标未央「空间锚点建基地 + 小行星带私掠 + 拉格朗日点/星门研究线」。

### 1.1 恒星与轨道带

| 区域 | 设定 | 写作用途 |
|------|------|----------|
| 主星「启明」 | 稳定 G/K 型主序星，辐射适中 | 基地默认轨道安全 |
| 内带 | 岩石小行星 + **金属**富集 | 开局采矿第一优先 |
| 中带 | 稀疏晶体矿 + 联络站级贸易航路萌芽 | 中期扩展 |
| 气巨伴星近轨 | **重氢**采集点（工程/槽位有限） | 曲率/机动燃料瓶颈 |
| 小行星乱域「锈环」 | 私掠出没（对标未央第一章私掠） | 清野与护航教学 |
| 外缘「考核星门工区」 | 人联/发包方预置的**教学星门空域**（未建成或半建成） | A2 控制目标；终局归属戏 |

### 1.2 拉格朗日点（玩法名，可保留）

- **L 点教学锚**：稳定曲率试验/前哨位（未央章节有「空间共振点」「高能试验装置」线，V01 只出现**简化版信标与观测点**）。
- 小说：人联在启明系预埋可测节点，考核「会不会抢节点、会不会护工程」。

### 1.3 通行规则

- 大船**物理可进**，但开局金属/晶体/重氢与指挥值不够支撑超主力海。
- 主力节奏：护卫墙 + 工程队 + 少量巡洋旗舰感（KCCP 系指挥值高、造价高，宜中后期）。

## 4. 日常循环（一章可写的最小闭环）

```
计划圈/前哨选点 → 工程采集（金/晶/氢）
    → 船坞造 FG300 / 修损
    → 护航清锈环或护运输
    → 情报更新矿点与敌情
    → 夜间结算：资源Δ、伤员、直播打赏(开拓币)
```

对标：

- 拉格朗日：采矿工程、计划圈经营（3.0）、维修与指挥值  
- SC：SCV 不停采、补建筑再作战  
- 群星：区划开发顺序

## 10. 胜利 / 失败（厚做确认版）

**胜（须同时）：**

1. 基地与工程循环存活至评定日；  
2. 对教学星门空域/指定观测节点有**可验证控制份额**（可联合）；  
3. 国家有效贡献 > 0；  
4. 完成人联公布的阶段清单（清指定私掠强度、矿产量阈值等——具体数写作时定，逻辑对标未央阶段任务）。

**败：** 基地被拔；核心编制归零；弃协议；零贡献混榜。
