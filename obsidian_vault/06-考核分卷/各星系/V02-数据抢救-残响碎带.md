# V02 · 残响碎带（数据抢救 / 考古）· 厚做

## 一句话

在 **舰骸与小行星碎片海** 中打捞遗产数据；大船难行，轻型编队 + 解析岗决定国运与编制印象。

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 拉格朗日 [数据抢救协议](https://wiki.biligame.com/wjdlglr/数据抢救协议) | 场景锚（协议已取消仍可作考核题型）；数据节点/遗迹逻辑 |
| 评定同构 | [[协议-未央开拓]]：目标控制 + 量化贡献 + 非零参评 |
| 物理 | 小行星带碰撞、遮挡、微陨石、局部辐射 |
| 航行通识 | [[拉格朗日点与曲率航行]] 碎片区规则 |
| 提取边界 | **环境杀伤 > 大会战**；虫潮级留给 V05 |

## 1. 星系地理

| 区域 | 设定 | 用途 |
|------|------|------|
| 残骸核「回响心」 | 大型舰骸/破碎站体团簇 | 高价值数据舱集中，辐射/结构不稳 |
| 碎石廊 | 密集小行星通道 | 大船禁入或极慢；护航艇/护卫主场 |
| 静默穴 | 低辐射避难凹坑 | 临时锚点、野修 |
| 拾荒环 | 中立拾荒 orth 私掠混居 | 交易黑匣 / 劫夺 |
| 交付锚「归档门」 | 人联指定的数据中继/半门设施 | **A1 提交点**（对标接收站逻辑） |

### 航行硬规则

1. **巡洋以上**在碎石廊曲率受限，建议拆轻编突入。  
2. 长距曲率易撞岩 → 多短跳 + 侦察开路。  
3. 撞击与辐射见 [[辐射真空与污染]]。

## 4. 数据节点规则（写作）

| 等级 | 内容感 | 风险 |
|------|--------|------|
| 白 | 日志、星图碎片 | 低 |
| 蓝 | 蓝图残段、技术参数 | 中；诱抢 |
| 金 | 人联点名「必交档案」 | 高；评定刚需 |
| 黑 | 污染/陷阱数据 | 接主网可炸（预演 V07） |

- **提交量 = 贡献**（同构特洛伊「提交=积分」）。  
- 未提交可自用研究，但评定不认。  
- 独吞金级且毁灭备份 → 可写国际丑闻 + A5 扣。
