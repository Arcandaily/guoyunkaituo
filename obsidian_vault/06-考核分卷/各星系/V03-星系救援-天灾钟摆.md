# V03 · 天灾钟摆（星系救援）· 厚做

## 3. 撤离链闭环

```
登记受护人口/资产 → 装载优先级排序
  → 护航至中转锚 → 补给/抢修
  → 窗口内冲撤离门廊 → 提交撤离证明（A1）
  → 留下断后编队（高 B2/B4 风险）
```

### 装载优先级（可改，但要公开）

1. 民用与伤员  
2. 关键数据/晶体样本  
3. 可再生产业模组  
4. 战斗舰（能走则走）  

弃民保船 = **B4/A5 重扣**，即使 A1 数字好看。

## 7. 胜负（确认）

**胜：** T-0 前，受护主体撤离率达人联指标；门廊证明有效；中转链未系统性瘫痪。  
**败：** 指标未达标且有可撤未撤；或链崩导致二次屠杀式伤亡。  
**灰胜：** 指标擦线但弃民证据确凿 → 国运可过、战友分崩、舆论战。
