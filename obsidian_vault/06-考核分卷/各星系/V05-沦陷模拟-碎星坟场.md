# V05 · 碎星坟场（数据抢救 · 沦陷模拟）· 厚做

## 一句话

**行星碎块 + 生物压境模拟库** 中抢救数据、守住临时防线；考的是「沦陷前夜」的组织力，不是开图鉴杀怪。

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 拉格朗日数据抢救场景 | 碎片导航、数据节点（V02 加强） |
| [[机制-泰伦虫潮]] 或 [[机制-异虫巢群]] | **二选一主虫**，避免混写 |
| [[V02-数据抢救-残响碎带]] | 碎带航行、归档提交同构 |
| SC/战锤生物战 | 火力纪律、检疫、突触/巢穴 |
| 通识 | [[生态与检疫]] [[拉格朗日点与曲率航行]] |

**本书默认主虫（可改，但要全书一致）：** 泰伦虫潮（噬潮）为主；异虫仅作旁支感染事件。
