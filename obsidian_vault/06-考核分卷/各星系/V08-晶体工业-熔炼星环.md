## 2. 星系地理

| 区域 | 设定 | 用途 |
|------|------|------|
| 外环采场 | 更大范围散脉 | 勘探岗高光 |
| 熔炼星环 | 零散小行星焊成的加工带 | **产线主体** |
| 热污染扇区 | 废热/辐射副产物 | 医疗/环评戏 |
| 质检闸+接收站 | 成品唯一正规出口 | A1 |
| 能源脊 | 重氢/反应堆阵列 | 产线「油」 |
| 星门仓储 | 终局外运 | A2 高阶 |

### 工业链（必须写清）

```
勘探圈定 → 工程开采 → 原矿库
  → 破碎/选矿 → 提纯（特晶锭）
  →（可选）元件预制
  → 质检闸 → 接收站提交（积分）
  → 废料处理（不可直排感染/爆炸）
```

缺任何一环，仓库堆山也不算工业国运。

## 8. 胜负（确认）

**胜：** 采-炼-交闭环在评定前稳定运转；成品（或等价提纯物）提交达指标；产线非一次性作秀。
