## 5. 双轨决胜细则

卷权重 **1.8**；子项：A1 0.15 / A2 0.15 / A3 0.15 / A4 **0.25** / A5 **0.20** / A6 0.10。  
战友：B4 **0.25**、B2 **0.25**、B5 升高。

### 国运 A（领袖）

| 子项 | 高分 | 一票伤 |
|------|------|--------|
| A4 | 战区击毁/守线贡献可审计 | 溃逃弃棱堡 |
| A5 | 与人联/友军协同、不背刺 | 卖友军坐标 |
| A2 | 决胜门/棱堡控制份额 | 零据点 |
| A3 | 前线后勤不断 | 弹药链断送友军 |
| A1 | 人联作战任务完成 | 假报战果 |
| A6 | 直播下的文明与勇气 | 纯作秀误事 |

**领袖算法（叙事）：**  
`国运总分 = Σ(卷分 × 卷权重)` → V10 拉高差距；前序垫底可用本卷翻盘但难反超灯塔级压制，除非 V10 爆发+前序不差。

### 战友 B（编制）

| 子项 | V10 铁律 |
|------|----------|
| B4 | 抗腐蚀、不献祭友军、不投邪 |
| B2 | 裂隙下不崩 |
| B5 | 防护/反虫/净网/登陆稀缺 |
| B6 | 人联观察员终评 |

**编制夜：** 高 B 者收预备役/猎人级邀约；低 B 高 A 国家仍可领袖，但「无人愿去」的孤岛结局。
