## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| **用户硬设定** | 进考核星系后 **全员即开拓者**（对标拉格朗日玩家） |

## 身份铁律

```
选人（蓝星）→ 传送 → 身份统一：「开拓者」
L0 现实职业 = 背景/技能源
L1 考核岗位 = 编队分工
```

**禁止再写**：某人「还不是开拓者、只是随船技工」之类降格。  
技工、舰长、矿头——**都是开拓者**，只是 L1 不同。


## 双目的

| 问题 | 轨道 | 看什么 |
|------|------|--------|
| 国家能否赢 | A 国运 | 开拓者编队 L1 是否完整、协议交付、升本进度 |
| 个人是否被要 | B 战友 | L0+L1 的胜任、协作、**守序**、稀缺思路 |
| 毒枭护运变态强 | A 可能涨 | B4 编制夜可能拒 |

## 选人原则（L0）

1. **不要十个程序员**：要职业光谱（见 14 个身份类）。  
2. **每国至少一个「刺」**：灰黑、偏执、资本、狂热（[[各国选人口味样本]]）。  
3. **AI 时代默认全员会用 AI**，差异在「职业问题意识」。  
4. **华国第 11 席**：L0 非常规（修科/边缘）+ 不占 L1 工程刚需位；**仍是开拓者**。  

## L1 骨架（每国开拓者团队仍须凑齐）

| 槽 | L1 职能 | 常见 L0 来源 |
|----|---------|--------------|
| 1 | 指挥 | 军官、企业家、公务员、特种军官 |
| 2–3 | 工程/采矿 | 矿工、工程师、电工、程序员 |
| 4–5 | 战斗/护航 | 特种、军人、警察、雇佣兵、帮派（险） |
| 6 | 情报 | 记者、警察、黑客、摄影师 |
| 7 | 科研 | 医生、科学家、程序员 |
| 8 | 后勤 | 供应链、会计、司机头、毒枭物流（险） |
| 9 | 医疗 | 医生、护士、心理 |
| 10 | 政工/外交 | 教师、神父、律师、网红公关、政工 |

灰黑 L0 可填 L1，但 **写明代价**。

## 禁止

- 把 L0/L1 写成「谁才有资格叫开拓者」。  
- 只写舰长不写地球职业。  
- 灰黑无代价英雄。  
- 歧视性刻板把某国写成「全是罪犯」。
