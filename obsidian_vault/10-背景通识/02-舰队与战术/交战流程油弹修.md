# 交战流程：油 · 弹 · 修

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 拉格朗日 | 结构值损伤、回港/维修、资源再造；冷却与装填感；工程与战斗分工 |
| SC | 打完修单位、补人口 |
| 战锤 | 弹药与舰损现实感（点到为止） |
| 群星 | 舰船维修时间、船坞 |

## 最小交战闭环（每场可套）

```
1 侦察定目标（情报）
2 曲率/常规抵达（耗重氢/时间）
3 站位展开：前排墙 / 中输出 / 后工程或投射
4 交战：结构值下降、可能炸舰
5 脱离：护航撤退或清场
6 回修：金属晶体+时间+船坞槽
7 补弹/补员：电子货币或库存
8 记账：资源Δ → Excel 流水；A4/B1 事件
```

## 「油弹修」对应

| 口语 | 游戏/设定 | 缺了会怎样 |
|------|-----------|------------|
| 油 | 重氢、曲率窗口 | 赶不到/逃不掉 |
| 弹 | 弹药、预制模块、电子货币维修 | 射两轮哑火 |
| 修 | 船坞、结构修复、医疗减员 | 有船不能战 |
