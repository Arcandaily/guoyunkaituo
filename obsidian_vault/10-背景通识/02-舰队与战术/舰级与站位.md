# 舰级与站位

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 拉格朗日 | 舰级：工程 / 护航艇 / 战机 / 护卫 / 驱逐 / 巡洋 / 战巡 / 航母 等；**前排·中排·后排**；武器直射/投射/特殊 |
| Wiki 舰船图鉴 | 1–6 星难度；未央系开局舰 |
| SC | 兵种前排坦克与远程输出 |
| 战锤 | 护航、主力舰、舰载机编队感 |

## 舰级定位（写作）

| 类型 | 角色 | V01 代表 |
|------|------|----------|
| 工程舰 | 采、建、修相关工程 | X8/X10/X20 |
| 护卫舰 | 廉价墙、护航 | FG300 系 |
| 护航艇/战机 | 填线、突防、防空 | CV-II / BR050 等（中后期） |
| 驱逐 | 中坚输出/功能 | AC721 等（非开局） |
| 巡洋 | 旗舰感、火力与载机 | CAS066、KCCP |
| 航母/战巡 | 超主力 | 高星；资源锁 |

## 站位（前中后）

- **前排**：吃伤害、掩护（装甲 FG300）。  
- **中排**：灵活输出/侦察。  
- **后排**：工程、投射、载机平台。  

交战时写清「谁在前挡、谁在后采/射」，读者才懂护航意义。

## 武器属性（简述）

- **实弹**（火炮/导弹/鱼雷）vs **能量**（离子/脉冲等）：克制可点到为止，避免每章公式。  
- 防空、对舰、攻城三分（拉格朗日面板）→ 占站要攻城/破甲感，清机要防空。
