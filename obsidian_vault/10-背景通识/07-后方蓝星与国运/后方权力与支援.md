# 后方权力与支援

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 本小说设定 | 国运争霸、国家支持团结 |
| 群星 | 帝国政令、联邦政治 |
| 战锤 | 帝国遥远指挥与补给线（官僚延迟） |
| 拉格朗日 | 枢纽/财团发包；援助包一次性，非常驻无限补兵 |

## 后方能做什么

| 支援 | 限制 |
|------|------|
| 政治授权、人选替换（若人联允许） | 考核封闭：不能随便空投军团 |
| 情报分析、舆论动员 | L3 延迟 |
| 技术预案、地球侧科研 | 未必赶得上当卷评定 |
| 外交施压他国后方 | 反噬谈判桌 |

## 后方不能做什么

- 直接改人联 **国运分/战友分** 账本。  
- 无视协议往考核系塞超规格主力（除非剧情特批且公开代价）。  

## 华国优势

- 举国体制：后方统一口径、资源预案快 → 写 A5 素材。  
- 灯塔：资本与科技院所并行，效率高但口径可能分裂。
