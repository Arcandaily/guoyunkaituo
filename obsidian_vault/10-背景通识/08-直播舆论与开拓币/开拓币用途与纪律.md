# 开拓币用途与纪律

## 灵感来源（必填）

| 来源 | 提取 |
|------|------|
| 本库 novel_mechanics | 开箱、购蓝图、灵能共振材料（主角特例） |
| 拉格朗日 | 技术黑箱、蓝图研究节奏 |

## 用途

1. 开箱（低/中/高级，作者可控）  
2. 直购蓝图碎片/技术模块  
3. 主角：少量兑换修科实验材（需库存与风险）  

## 纪律

| 规则 | 原因 |
|------|------|
| 个人打赏默认个人账户 | 激励高光 |
| 大战前可决议抽成公库 | A5 团结戏 |
| 禁止把币当军饷买人命 | 道德与 B4 |
| 账目可审计 | 后勤/政委 |
